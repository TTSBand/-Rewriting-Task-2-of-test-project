RDP Control Settings

Purpose:

RDP Control enables you to configure RDP settings and display window parameters. Options default and rdp5_only are set by default but you can create custom settings profiles, which you can rename for identification later.

Configuring RDP Control Settings

Steps:

1. Navigate to Main menu > RDP Control > Settings .

2. Click + in the lower right corner of the screen to create a new custom settings profile.

|Tip: If you have multiple settings profile tabs, click IKON to close irrelevant tabs.

3. Enter information into the red rectangular empty field ( ) to name the settings for identification later, for example .. user_office_secure_1.

|Tip: Use informative names that refer to user/admin/group/security level, and so on.

4. Configure the settings below:

  • 4.1 To set idle length time, enter a value larger than 1 into the Idle timeout field.

    • |CAUTION: Consider the value in milliseconds when entering a number.

  • 4.2 To set the maximum display width value for the RDP window, enter enter a value larger than 1 into the Maximum display width field.

  • 4.3 To set the maximum display height value for the RDP window, enter a value larger than 1 into the Maximum display depth field.

  • 4.4 To select the RDP version you want to allow, select either or more of the Enable RDP 4 / Enable RDP 5 / Enable RDP 6 or newer options

  • 4.5 To overcome potential compatibility problems, select Enable RDP 4-style authentication

  • 4.6 To enable performing an evaluation before connection is established, select Enable pre channel check

  • 4.7 To reduce the risk of information overload during RDP sessions, select Enable compression

  • 4.8 To change the default -- - AUTO suffix, enter the approppriate suffix into the Autologon domain suffix field.

  • Click the [COMMIT] action button to save the changes to your settings. You can create more of these custom settings profiles by repeating the procedure above.

  • You can later select the custom settings profile you wish to use by navigating to Main menu > RDP Control > Connections.
